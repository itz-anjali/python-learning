Program ko chhote hisson (Threads) mein todna taaki wo parallel (saath mein) chal sakein.
Kab use karein?

Jab Waiting zyada ho (e.g., File Downloading, Database se data lana).

Note: Heavy calculations ke liye Python mein Multithreading slow ho sakti hai (GIL ki wajah se).

. Main Commands (import threading):

Thread banana: t = threading.Thread(target=function_name)

Start karna: t.start() (Ab ye background mein chalega).

Wait karna: t.join() (Main program rukega jab tak thread khatam na ho).
---------------------------------------------------------------
